# 10000-swords-against-heaven-CSB
Custom System Builder compendium for the TTRPG 10000 swords against heaven

You'll need the Custem system Builder system for FoundryVTT, (https://foundryvtt.com/packages/custom-system-builder)
This currently contains 2 real things. 

Custom character sheet
Dice Roll macro for success/failure, compatible with Dice so nice. 

I aim to add weapon attacks, weapons, etc, as i decide the best way to roll them. Those must be resolved manually for now.
